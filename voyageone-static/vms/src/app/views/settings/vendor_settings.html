Rule of Shipment Naming Converter

We can show you a default name when creating a new shipment with the rules here.
  • Add [] around plain text.

    [TestName] -> TestName

    [YYYY-MM-DD] -> YYYY-MM-DD

  • We takes a string of tokens and replaces them with their corresponding values.

    YYYY-MM-DD -> 2016-08-31

    YYYY-MM-DD HH:mm:ss -> 2016-08-31 10:20:20

    [ShipmentTestName_]YYYY-MM-DD -> ShipmentTestName_2016-08-31
